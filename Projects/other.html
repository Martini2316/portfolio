Repozytorium Online

Repozytorium Online dla mojej orkiestry. Stworzyłem je podczasz pandemi, żeby każdy muzyk naszej orkiestry, miał pod "ręką" wszystkie potrzebne nutki.

Gra Magiczny Kamień ( GameJam )

Moja gra, dzięki której wygrałem szkolny konkurs GameJam. Jest napisana w czystym JavaScript i pokonała ona inne drużyny(od 1 do 5 osób), które używały silników do tworzenia gier!

Gra waży zbyt dużo aby udostępnić ją na github pages, wieć odsyłam do nagrania na YouTube, gdzie jest pokazane działanie gry.

Jeśli masz ochotę zagrać, napisz do mnie, a udostępnie ci link do pobrania! :)
